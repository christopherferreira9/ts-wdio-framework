# Typescript Webdriver.io POM framework


## How to run:

### Clone the repository
```
git clone https://github.com/christopherferreira3/ts-wdio-framework
```

### Install dependencies
```
npm install
```

### Run the tests
```
BROWSER_NAME=firefox npm run test:ci
```
